Project Documents

This page presents the complete document collection of TruGanic, covering academic, technical, and submission-level files used throughout the project lifecycle. It includes the project charter, project TAF, project proposals, final reports, log books, and the research paper. Together these files show how the project was planned, reviewed, executed, and finalized from start to end. Each section below gives a quick summary so users can understand what that document set contains before opening it. This helps supervisors, evaluators, and team members quickly find the right evidence for planning, progress, and final outcomes. Use the View button to open each file directly and the Download button to save a copy for offline reference or submission.
